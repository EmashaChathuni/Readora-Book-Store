# Book Store App - Sri Lankan Edition

## Quick Start

This Flutter app is designed for Sri Lankan book lovers, featuring a curated collection of Sinhala classics and contemporary Sri Lankan literature.

## Featured Authors

### Sinhala Literature
- **Martin Wickramasinghe** - Gamperaliya trilogy, Madol Duwa, Viragaya
- **Kumaratunga Munidasa** - Language reformer and essayist
- **G.B. Senanayake** - Rural fiction

### Contemporary Sri Lankan Authors (English)
- **Shehan Karunatilaka** - Booker Prize winner (The Seven Moons of Maali Almeida)
- **Michael Ondaatje** - International bestselling author
- **Shyam Selvadurai** - LGBTQ+ and cultural themes
- **Romesh Gunesekera** - Shortlisted for Booker Prize
- **Nayomi Munaweera** - Epic historical fiction

## Book Categories

1. **Sinhala Classic** - Timeless Sinhala literature
2. **Sinhala Fiction** - Contemporary Sinhala novels
3. **Sinhala Essays** - Cultural and philosophical writings
4. **Sri Lankan Fiction** - English language novels by Sri Lankan authors
5. **Memoir** - Personal narratives and family histories

## Pricing

All prices are displayed in **Sri Lankan Rupees (LKR)**
- Format: Rs. 1,250.00
- Price range: Rs. 600 - Rs. 1,500

## Technical Details

- **Frontend**: Flutter
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Currency**: Sri Lankan Rupees (LKR)
- **Language Support**: English interface, Sinhala book titles

## Installation

```bash
# Get dependencies
flutter pub get

# Run the app
flutter run
```

## Adding Custom Books

To add more Sri Lankan books, update `lib/utils/sample_data.dart` with:
- Title (in Sinhala or English)
- Author name
- Description
- Price in LKR
- Category
- Rating (0-5)
